Política de Privacidad

Inicio

1. Datos que Recopilamos

Recopilamos el ID de usuario, ID de servidor y configuración de seguridad para crear y gestionar copias de seguridad.

No almacenamos contenido de mensajes ni información privada de los usuarios.

2. Uso de los Datos

Los datos solo se utilizan para proporcionar la funcionalidad de copias de seguridad.

No se venden, comparten ni utilizan con fines comerciales.

3. Seguridad y Protección

Implementamos medidas avanzadas de seguridad para proteger los datos.

No garantizamos seguridad absoluta, pero tomamos todas las precauciones necesarias.

4. Eliminación de Datos

Los administradores pueden solicitar la eliminación de copias de seguridad en cualquier momento.
